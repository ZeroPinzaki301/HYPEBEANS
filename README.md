# ☕ HYPEBEANS
### Online Ordering & Inventory System with Map API Integration

**IPT101 Project** | MERN Stack | Hosted on Render

---

## 📌 About

HYPEBEANS is a web-based system for coffee shop online ordering and inventory management. Customers can browse products, place orders, and locate the store via integrated map API. Admins can manage products, track inventory, and process orders.

## 🛠️ Tech Stack

- **MongoDB** - Database
- **Express.js** - Backend framework
- **React** - Frontend library
- **Node.js** - Runtime environment
- **Google Maps API** - Location services
- **Render** - Hosting platform

## ✨ Features

- User authentication (Login/Register)
- Product browsing with categories
- Real-time inventory tracking
- Shopping cart & order placement
- Order status tracking
- Google Maps integration for store location
- Admin dashboard for inventory management
